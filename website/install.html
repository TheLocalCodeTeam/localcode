Install

Get Localcode

Open source. Local-first. 139 agents. Ready in 30 seconds.

Global install (recommended)
npm install -g @localcode/cli
Installs the localcode command globally.
No install needed
npx @localcode/cli
Runs directly via npx. No global install required.
1

Install

Run the install command above. Node.js 18+ is required.

2

Launch

Run localcode in your terminal. The setup wizard will guide you through provider selection.

3

Choose your provider

Pick Ollama for free local AI, or connect OpenAI, Anthropic, or Groq with your API key.

4

Start coding

Describe what you need. Localcode reads your codebase, makes changes, runs tests, and ships.
