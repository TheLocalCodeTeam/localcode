Features

Everything you need.
Nothing you don't.

Localcode packs every feature a serious developer needs — plus the things cloud assistants will never give you.

◉

100% local with Ollama

Run any Ollama model — qwen2.5-coder, deepseek-r1, llama3.2, and more. No API key. No cloud. Your code never leaves your machine. Free forever.

Free forever
🤖

139 specialized agents

From AI engineers to security auditors, database optimizers to DevOps automators. Each agent has deep domain expertise and activates automatically based on task context.

Auto-dispatch
🌐

Multi-agent orchestration

NEXUS pipeline coordinates multiple agents through 7 phases with quality gates, Dev↔QA loops, parallel execution, and automatic retries. Full/sprint/micro modes.

NEXUS
⟳

Switch provider live

Use /provider mid-session. Switch from local Ollama to GPT-4o to Claude without restarting. API keys stored per-provider and auto-loaded from environment variables.

Multi-provider
⧉

Plugin system

Drop a .js file in ~/.localcode/plugins/ to add custom slash commands. Full access to conversation context and tools. Zero boilerplate — just name, description, and run function.

Extensible
⚡

Hooks

PreToolUse, PostToolUse, and Notification hooks — run custom scripts before or after any tool call. Wire up linters, formatters, notifications, or anything else.

Automation
◈

Memory files

.localcode.md for global memory, .localcode.md in project root for project context. Both loaded automatically at startup. Persistent context across sessions.

Persistent context
⬡

MCP servers

Connect Model Context Protocol servers — both stdio and HTTP transports supported. Give Localcode access to your databases, APIs, and external tools.

Extensible
⊟

Vision input

/image path/to/screenshot.png — works with Claude, GPT-4o, and Ollama llava. Describe UI, debug layouts, analyze diagrams.

Multimodal
⎇

TF-IDF codebase search

/index builds a full semantic index of your project. Ask Localcode to find relevant files without passing your entire codebase as context.

Smart context
⚿

Permission prompts

Every file write, patch, or shell command asks first. Press y, n, or a (allow all). Full control, always. Configurable per-tool and per-command pattern.

Safety
🔒

Budget guard & safe mode

Set a USD budget limit — auto-switches to local Ollama when hit. Safe mode with git stash before edits and auto-revert on test failure.

Protection
